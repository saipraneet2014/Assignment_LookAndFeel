Data Model

Data Source and Description

We have used National Renewable Energy Laboratory application programming interface. All the api documents and usage guides are available here.

From the dataset, we can get the nearby fuel stations in the city along with the information about the station based on the type of fuel available.

Data Model

The data model contains one one-many relationship and one many-to-many relationship. One between the entity Fuel_Stations and EV and another between Fuel_Stations and Federal_Agency. This many-to-many realtionship is converted into a one-to-many relationship for easy schema understanding.
